# MyBox

<center><img src="/img/portfolio/MyBox_main.png" alt="MyBox_main.png" width="100%"></center><br>  
  
MyBox는 집 안의 모든 보관함에 저장된 물건들을 정리하기 위해 디자인된 앱입니다. 이 앱을 사용하여 각 보관함에 어떤 물건들이 있는지 저장할 수 있으며, 이를 이름과 키워드를 사용하여 쉽게 검색할 수 있습니다. 검색된 물건을 누르면 그것이 어떤 보관함에 보관되어 있는지 확인할 수 있습니다.

MyBox는 집에 어떤 물건을 가지고 있는지 일일이 다 기억하지 못해서 생기는 문제를 해결하기 위해 만들어졌습니다.(중복 구매나 물건을 잃어버리는 일 등)  
MyBox를 사용하여 집 안의 모든 보관함을 체계적으로 정리하고 관리하세요.

🛠️ 기술 스택: SwiftUI, MV(Model-View)
